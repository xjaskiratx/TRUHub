# TRUHub
TRUCompanion is a comprehensive mobile application designed to support Thompson Rivers University (TRU) students in navigating campus life, local services, communities, and essential university resources. Built using modern Android development practices, the app offers a user-friendly experience with adaptive navigation, contextual content feeds, and a focus on discoverability.
Key Features
Explore Campus Services – Browse academic, administrative, and student support services.
Community & Clubs Directory – Discover student communities, clubs, and ways to get involved.
Events & Activities – Stay up to date with campus and local events tailored for TRU students.
City Resource Guide – Access information on nearby places, transit, and essential local amenities.
Dynamic Navigation – Multi-layered UI structure that guides users from broad categories down to specific details.
Responsive & Intuitive Design – Clean interface optimized for mobile usability and engagement.
